# Design System — Referência GeoMAG

Pasta de referência visual. **Estes arquivos não fazem parte do build** — são consulta de design para guiar implementação.

## Arquivos

| Arquivo | Tema | Uso |
|---|---|---|
| `geomag-design-system-dark.html` | Escuro (principal) | Paleta, tipografia e componentes no tema dark |
| `geomag-design-system-light.html` | Claro | Variações de cor e contraste no tema light |

## Como usar

Abra os HTMLs diretamente no browser para visualizar:

```
design-system/geomag-design-system-dark.html
design-system/geomag-design-system-light.html
```

## O que extrair daqui

- **Cores**: tokens de cor (primária, secundária, neutros, feedback)
- **Tipografia**: famílias, pesos e escala de tamanhos
- **Espaçamento**: grid e spacing tokens
- **Componentes**: botões, cards, inputs, badges — como referência de aparência, não de código
